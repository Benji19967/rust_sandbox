 ## `cargo`

`cargo build`

builds a binary in `target`

 `cargo run`

 builds a binary in `target` and runs it

 `cargo clean`

 removes `target` directory

 `cargo install`

 installs the binary under `~/.cargo/bin`. Example: `cargo install --path .`

 `cargo uninstall`

 uninstall the binary from `~/.cargo/bin`
